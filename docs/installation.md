# Installation

## Requirements

- Docker and Docker Compose
- Node.js and npm
- Python 3
- PostgreSQL client tools (optional)
- Redis client tools (optional)

## Setup

1. Clone repository:

```bash
git clone https://github.com/your-org/zttato-platform
cd zttato-platform
```

2. Start platform bootstrap:

```bash
./start-zttato.sh
```
